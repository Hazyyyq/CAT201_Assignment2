# CAT201_Assignment_2
A repository for CAT201 Assignment 2, Introduction to our company static (or dynamic) webpage.  
This repository will be used as a git control to keep track of each commit and update.

## Run Locally
To run the project on your localhost:

1. Clone the repository or Download as ZIP files:
2. Install dependencies via terminal:
`   npm install`
3. Change the directory via terminal:
`   cd react-frontend`
4. Run the dev via terminal:
`   npm run dev`
5. Click the link of the local host
6. Enjoy the Kaki Gamerz website
